# Full-Stack E-Commerce Web Application

A **full-stack e-commerce web application** developed as a course project for **CS308 – Software Engineering** at **Sabancı University**.  
The system simulates an online store with **role-based access**, **product management**, **order processing**, and **user interactions**, built using a **Spring Boot backend** and a **modern JavaScript frontend**.

---

## Features

### User Features
- Browse, search, and sort products  
- Shopping cart for both guests and authenticated users  
- User authentication and checkout flow  
- Order placement and order history  
- Order status tracking and invoice viewing  
- Product ratings and comments  
- Order cancellation and return requests based on order status  

### Management Roles
- **Role-based authorization** (Customer, Sales Manager, Product Manager)  
- Product, category, and stock management  
- Order delivery management and status updates  
- Comment moderation workflow  
- Discount and pricing management  
- Revenue and profit analysis over selected date ranges  

---

## Tech Stack

### Backend
- **Java**
- **Spring Boot**
- **RESTful APIs**

### Frontend
- **JavaScript**
- Modern frontend framework (**Vite-based development server**)

### Database
- **Relational database** for persistent storage of users, products, orders, and invoices

---

## Running the Project

### Backend
```bash
mvn spring-boot:run
```

### Frontend
``` bash
npm install
npm run dev
```
